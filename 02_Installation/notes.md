## Installation

1. Need Node.js (To Check Node.js Version `node -v`)

2. Need NPM (To Check NPM Version `npm -v`)

3. run `npm install -g typescript` (Intall Globally)

4. To Check Typescript Version `tsc -v`




## How To Write Typescript in File

1. Create File `filename.ts` with .ts extension. .ts its a Typescript File

2. Run `tsc filename.ts`

3. It Create Same Name File with .js extension

4. Run `node filename.js`
